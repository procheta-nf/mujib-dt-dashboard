Dashboard Overview
Mujib Basin Digital Twin
Eco‑Hydrological Decision Support

This dashboard is a prototype eco‑hydrological digital twin for the Mujib Basin. It combines spatial layers, hydrological model outputs and operational climate information to support exploration, comparison and scenario‑based discussion for water and land management.

What you can do
  • Explore basin layers (boundary, rivers, dams, stations, buildings).
  • Compare vegetation change using NDVI (baseline, recent, Δ).
  • Review SWAT outputs for sub‑basins and compare interventions/scenarios.
  • Check ERA5 precipitation and ERA5‑Land soil moisture context.
  • Use the mini weather widget for short‑term forecast awareness.
How to start
  1. Turn on Basin Boundary (default) to orient yourself.
  2. Open a category in the left panel and toggle layers as needed.
  3. Click a sub‑basin to populate the analysis panel and charts.
  4. Switch chart tabs (Hydrology / Climate / Soil) for details.

NDVI
Baseline NDVI (2015-2017) ℹ
Baseline NDVI (2015-2017): typical vegetation greenness during baseline period.
The baseline map represents natural vegetation conditions before recent changes. Within this dataset (range ≈ 0.04-0.22), higher values correspond to relatively denser vegetation, while lower values indicate bare soil or sparse vegetation.
Recent NDVI (2023-2025) ℹ
Recent NDVI (2023-2025): current vegetation greenness.
This recent map shows current vegetation conditions. Compare it to the baseline to spot increases or decreases in greenness. Within this dataset (range ≈ 0.05-0.27), higher values correspond to relatively denser vegetation, whereas lower values indicate bare soil or sparse vegetation.
Δ NDVI (2023-2025 − 2015-2017) ℹ
Δ NDVI (Recent − Baseline): change over time.
The Δ NDVI map highlights vegetation change between the baseline and recent periods. Positive values (greater than +0.01) indicate vegetation increase, negative values (less than −0.01) indicate vegetation decrease, and values near zero (between −0.01 and +0.01) indicate little or no change.
NDVI value meaning:
0.04-0.06: very sparse / bare soil
0.06-0.08: low vegetation
0.08-0.10: sparse vegetation
0.10-0.13: moderate vegetation
> 0.13: relatively high vegetation
Note: NDVI values are scaled to the observed range (vmin-vmax) for each period.
Δ interpretation: Increase > +0.01 • Decrease < −0.01 • Stable −0.01 to +0.01

        What is CMIP6?
        CMIP6 (Coupled Model Intercomparison Project Phase 6) provides standardised global climate model projections used by the IPCC. SSP2-4.5 represents an intermediate emissions pathway; SSP5-8.5 a high-emissions scenario. Values are proxy perturbations calibrated to published regional projections for Jordan (Alsalal et al. 2024), scaling ERA5 precipitation and temperature to simulate possible future conditions.
